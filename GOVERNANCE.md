# Governance

## Project ownership

AgentSeatbelt is currently maintained by Moosa Alam.

## Decision model

- Security and trust-impacting changes require deterministic behavior and tests.
- Breaking CLI behavior changes must be documented in `CHANGELOG.md`.
- Public release notes are tracked in `RELEASE_NOTES_v0.1.0.md` (and future versioned notes).

## Contribution standards

- Follow [CONTRIBUTING.md](CONTRIBUTING.md).
- Keep PRs scoped and reviewable.
- Preserve local-first, no-telemetry guarantees.
